비교연산자를 여러 개 사용해 복잡한 조건식을 만드는 연산자입니다.

| 연산자 | 설명                                                                      |
| ------ | ------------------------------------------------------------------------- |
| and    | 왼쪽, 오른쪽 모든 비교 연산의 결과가 참(true)일 때 참(true)를 출력        |
| or     | 왼쪽, 오른쪽 비교 연산 결과가 어느 하나라도 참(true)일 때 참(true)을 출력 |

```python
id = input('아이디 입력 :')
if id=="admin" or pw=="super" :
    print("로그인 완료")
else :
	print("로그인 실패")
```

```python
id = input('아이디 입력 :')
pw = input('패스워드 입력 :')
if id=="admin" and pw=="1234" :
    print("로그인 완료")
else :
	print("로그인 실패")
```
